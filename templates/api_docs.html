Service Provider Directory API
1.0.0

This is the OpenAPI-style documentation for the Public Service Provider Directory API. Below you will find all available endpoints, parameters, and response structures.

Providers

GET /providers Returns all service providers

Retrieves a comprehensive list of all professionals available in the directory.

Responses

[
  {
    "id": 1,
    "name": "Alvison Hunter",
    "trade": "Senior Web Developer",
    "phone": "+505 8863 8751",
    "email": "alvison@gmail.com",
    "photo_url": "...",
    "location": "Nicaragua",
    "experience_years": 6,
    "is_verified": true,
    "starting_rate": 70.0,
    "bio": "..."
  }
]
POST /providers Add a new provider

Creates a new service provider profile in the directory.

Request Body

{
  "name": "string (required)",
  "trade": "string (required)",
  "phone": "string",
  "email": "string",
  "photo_url": "string (url)",
  "location": "string",
  "experience_years": "integer",
  "is_verified": "boolean",
  "starting_rate": "float",
  "bio": "string"
}

Responses

201 Created 400 Bad Request
GET /providers/{pro_id} Find provider by ID

Returns a single provider's detailed information.

Parameters

NameDescription
pro_id *Integer ID of the provider

Responses

200 OK 404 Not Found
PUT /providers/{pro_id} Update an existing provider

Updates the details of an existing provider.

Parameters

NameDescription
pro_id *Integer ID of the provider

Request Body (Partial updates allowed)

{
  "name": "string",
  "trade": "string",
  "phone": "string",
  "bio": "string"
}
DELETE /providers/{pro_id} Delete a provider

Deletes a provider and gracefully cascades the deletion to all their associated reviews.

Parameters

NameDescription
pro_id *Integer ID of the provider

Reviews

GET /providers/{pro_id}/reviews Get reviews for provider

Retrieves all reviews associated with a specific provider.

Responses

[
  {
    "id": 1,
    "pro_id": 1,
    "rating": 5,
    "comment": "Great work!"
  }
]
POST /providers/{pro_id}/reviews Submit a review

Submit a new review for a provider.

Request Body

{
  "rating": "integer (1-5, required)",
  "comment": "string"
}
